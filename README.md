# ToDo using MERN

To-DO List App

1. Clone the repo 
    ```
    git clone https://github.com/Yaswanthkumar1100/To-do
    ```

2. Cd
    ```
    cd To-do
    ```

3. Install required packages
    ```
    npm install 
    ```
  
4. Cd to client side directory
    ```
    cd client
    ```
5. Start Server in client side
    ```
    npm start
    ```
    or
    ```
    npm run start 
    ```

5. Cd to server side directory
    ```
    cd server
    ```

6. Run the server in Server side
    ```
    nodemon index.js
    ```
